What have I done so far?

I have listed my 6 most important and hardest projects below. If you want to view what is in backend, just click on either the project name or icon. Note: All of the descriptions in my GitHub are in Turkish language. You can use translate when necessary.

Login System - Username and Password Checker

This program checks whether the username and password that was taken from user are both valid. It is not only very detailed and extensive, but also convenient program for you.

Hotel Reservation System

With this software, you will be able to book a hotel in Turkey, Antalya by seeing a list of hotel names that I used web scraping (BeautifulSoup4) to acquire data. The software also asks you create an account with a captcha in it.

Coffee and Food Automation

Users can list either coffees or food according to their decisions. I used Web Scraping (BeautifulSoup4) in this project for the second time. I acquired all data from Starbucks' website.

Tax Included Price Calculator

With this automation, firstly, you can learn the tax-free price and then tax included price of the item you chose to purchase. There are finite items that I have added.

Internet Speedtester

Users can not only calculate their download and upload speed and ping to the best server but also see where the best server is located. I used speedtest module (Python 3) in this project.

Send Whatsapp Message Automatically

You can send any message you want either immediately or delayed to any person you want on Whatsapp Web. Pywhatkit module is used in this project.
